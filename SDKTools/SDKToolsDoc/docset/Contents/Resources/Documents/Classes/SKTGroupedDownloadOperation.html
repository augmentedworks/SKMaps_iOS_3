SKTGroupedDownloadOperation Class Reference

Inherits from NSObject
Conforms to SKTGroupedOperationsDelegate
Declared in SKTGroupedDownloadOperation.h

Overview

SKTGroupedDownloadOperation contains multiple SKTGroupedOperations. For example a map download will contain 3 SKTGroupedOperations (Map, Name browser zip, texture file).
