downloadProgressDataForGroupedDownloadOperations:

Returns progress data for a set of SKTGroupedDownloadOperation objects.

+ (NSDictionary *)downloadProgressDataForGroupedDownloadOperations:(NSArray *)array

Parameters

array

NSArray of SKTGroupedDownloadOperation objects

Return Value

The dictionary contains following keys/values: – an NSString object under the kSKTDownloadOverallProgressString key, representing overall progress string for all SKTGroupedDownloadOperations. – an NSNumber object under the kSKTDownloadOverallPercentage key, representing overall percentage as a float value for all SKTGroupedDownloadOperations. – an NSNumber object under the kSKTDownloadTotalDownloadSize key, total download size as long long for all SKTGroupedDownloadOperations. – an NSNumber object under the kSKTDownloadTotalBytesDownloaded key, total downloaded bytes as long long for all SKTGroupedDownloadOperations.

Discussion

Returns progress data for a set of SKTGroupedDownloadOperation objects.
